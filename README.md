# 舌尖
## 项目结构
    -- src
        - assets
            图片、音乐、视频素材
        - components
            Component.vue，各页面需要的各组件封装
        - js
            Page.js，Page页面需要的各业务请求逻辑
        - pages
            Page.vue，Page页面模板
        - router
            index.js，路由表
        - store
            index.js，Vuex
        App.vue 根组件，进行路由、Vuex等全局注册
        main.js 入口
        JS.js 全局数据与业务对象
## 运行前请执行
- npm install
